En este tema, nos adentramos en el uso de CSS para dar estilo y estructura a nuestras páginas web. En el nivel básico, aprendemos a seleccionar elementos con selectores, definimos colores, fuentes, márgenes y rellenos, y comprendemos cómo funciona el modelo de caja para organizar el contenido. Cuando avanzamos al nivel avanzado, exploramos técnicas de posicionamiento, utilizamos flexbox y grid para diseñar layouts responsivos y creamos animaciones sencillas. También nos familiarizamos con pseudoclases y pseudoelementos para refinar detalles visuales. Gracias a estas prácticas, logramos interfaces atractivas y bien estructuradas que mejoran la experiencia de quienes visitan nuestros sitios.

¿Qué aprendimos?

Aprendimos a manejar los fundamentos de CSS para dar estilo a nuestras páginas web. Conocimos los distintos selectores y cómo la especificidad determina qué reglas se aplican. Trabajamos con propiedades de texto y fuentes para controlar tipografías, ajustamos márgenes, rellenos y bordes dentro del modelo de caja, y definimos colores y fondos usando valores en diferentes unidades (px, %, em, rem, vh, vw). También exploramos posicionamiento estático, relativo, absoluto, fijo y sticky, y nos iniciamos en Flexbox para crear diseños flexibles y en Grid Layout para distribuir elementos en cuadrículas. Finalmente, practicamos con pseudo-clases (por ejemplo, :hover, :active) y pseudo-elementos (::before, ::after), y agregamos transiciones y animaciones básicas para mejorar la interacción visual.

¿Cómo lo aprendimos?

Realizamos ejercicios prácticos en los que aplicamos selectores específicos para estilos de texto y fondos, visualizando resultados en el navegador en tiempo real. Construimos ejemplos de cajas con diferentes márgenes y rellenos para entender el modelo de caja, inspeccionando sus dimensiones con las herramientas de desarrollo. esarrollamos layouts sencillos con Flexbox y Grid, comenzando con plantillas guía y luego modificándolas para experimentar con justificadores, alineaciones y áreas de grid. Implementamos pseudo-clases en listas de navegación para observar efectos visuales al pasar el cursor, y diseñamos animaciones sencillas con keyframes y transiciones CSS para verlas reproducirse en el navegador.
