Frontend: El frontend es la parte de una aplicación web que interactúa directamente con el usuario. Incluye todo lo que se ve en pantalla, como el diseño, los colores, las fuentes, los botones y la navegación. Su objetivo es ofrecer una experiencia visual clara e intuitiva, utilizando tecnologías como HTML para la estructura, CSS para los estilos y JavaScript para la interactividad. Además, se apoya en frameworks modernos como React, Vue o Angular para facilitar el desarrollo de interfaces dinámicas y responsivas.

Backend: El backend es la parte interna de una aplicación web que se encarga de procesar la lógica del sistema, gestionar las bases de datos, autenticar usuarios y responder a las solicitudes del frontend. Funciona en el servidor y no es visible para el usuario final. Se desarrolla con lenguajes como Node.js, Python, PHP o Java, y suele utilizar frameworks como Express, Django o Laravel.

¿Qué aprendimos?

Aprendimos la diferencia entre frontend y backend en el desarrollo web. Entendimos que el frontend es la parte visual con la que interactúa el usuario, mientras que el backend es la parte interna que gestiona la lógica, los datos y la comunicación con el servidor.

¿Cómo lo aprendimos?

Lo aprendimos mediante una explicación teórica acompañada de ejemplos prácticos y comparaciones. Analizamos las funciones, tecnologías y responsabilidades de cada parte, lo que nos permitió comprender cómo trabajan juntas para crear una aplicación web completa.
