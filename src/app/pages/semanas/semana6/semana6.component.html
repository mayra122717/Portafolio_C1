En este tema aprendemos a diseñar y desarrollar sitios web modernos combinando tres herramientas clave. Con Bootstrap utilizamos rejillas y componentes predefinidos para crear páginas responsivas sin escribir CSS desde cero. Con Tailwind CSS aplicamos clases utilitarias en el HTML para personalizar estilos y optimizar el rendimiento eliminando código innecesario. Finalmente, con JavaScript básico adquirimos las bases para interactuar con el DOM, manejar eventos y validar formularios, dotando de interactividad a nuestras páginas.

¿Qué aprendimos?

En estos tres temas adquirimos las habilidades para diseñar interfaces web modernas y funcionales: con Bootstrap aprendimos a usar su sistema de rejillas y componentes preconstruidos para crear rápidamente páginas responsivas; con Tailwind CSS descubrimos el enfoque de utilidades atómicas, personalizando colores y estilos directamente en el HTML y optimizando el rendimiento con PurgeCSS; y con JavaScript básico aprendimos a escribir scripts que manipulan el DOM, validan formularios y gestionan eventos para añadir interactividad.

¿Cómo lo aprendimos?

Lo hicimos combinando teoría y ejemplos prácticos: confeccionamos prototipos en los que aplicamos clases de Bootstrap y Tailwind para ajustar diseños en distintos tamaños de pantalla, y escribimos pequeños programas en JavaScript que respondían a acciones del usuario, integrando así estilos y lógica en proyectos reales que reforzaron nuestra comprensión de cómo estos tres pilares trabajan juntos para crear sitios web eficientes y atractivos.
