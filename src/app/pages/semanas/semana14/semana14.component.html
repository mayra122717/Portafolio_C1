El documento es una guía teórico-práctica orientada al aprendizaje del desarrollo web utilizando Python y Flask, complementado con conceptos clave de configuración de entornos de servidor y herramientas auxiliares como Apache, mod_wsgi, MySQL, Jinja2 y PHP.

¿Qué aprendimos?

Durante este proceso de aprendizaje, adquirimos conocimientos fundamentales sobre el lenguaje de programación Python y su aplicación en el desarrollo web utilizando el framework Flask. Aprendimos desde conceptos básicos como sintaxis, estructuras de control, funciones y programación orientada a objetos, hasta aspectos más avanzados como la conexión a bases de datos MySQL, la gestión de rutas y formularios en aplicaciones web, y la integración con el motor de plantillas Jinja2.

¿Cómo lo aprendimos?

  • Aprendizaje guiado y práctico:
    • Seguimos guías paso a paso para instalar y configurar Python, Flask, Apache y MySQL.
    • Usamos pip para instalar librerías como flask y pymysql.
  • Laboratorios de codificación:
    • Desarrollamos una app en consola usando clases Persona y Estudiante.
    • Construimos una app web en Flask con formulario HTML y conexión a MySQL.
  • Ejercicios aplicados:
    • Aplicamos conceptos como rutas Flask app.route, manejo de formularios (request), y plantillas (render_template).
    • Utilizamos Jinja2 para crear vistas dinámicas con condicionales y bucles.
  • Consulta de recursos técnicos:
    • Estudiamos la documentación oficial de Flask y Python.
    • Reforzamos buenas prácticas con la guía de estilo PEP-8 y estructura MVC.
