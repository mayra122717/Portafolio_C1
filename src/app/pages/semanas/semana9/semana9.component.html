Se habló sobre las arquitecturas de aplicaciones web, servidores, y herramientas de desarrollo. Se mostró varias secciones que abordan desde la arquitectura cliente-servidor hasta la gestión de nombres de dominio y la configuración de servidores web. También los pasos para la instalación de herramientas como JDK, IntelliJ IDEA y Tomcat, así como el uso de Maven para la gestión de proyectos.

¿Qué aprendimos?

Se aprendió sobre las diversas arquitecturas web, incluyendo la arquitectura cliente-servidor clásica, multicapa, y las aplicaciones de una sola página (SPA), así como la arquitectura hexagonal que permite una mejor gestión del software. También se exploraron los tipos de servidores web y su configuración, los diferentes servicios de hosting y sus características, así como el concepto de cloud hosting y cloud computing. Además, se profundizó en la gestión de nombres de dominio, la configuración de servidores web, y se introdujeron herramientas de desarrollo como Maven y Gradle, junto con el funcionamiento de la autenticación basada en tokens.

¿Cómo lo aprendimos?

Se aprendió a través de una combinación de lectura teórica y práctica, donde el documento proporcionó explicaciones claras y estructuradas sobre cada tema, facilitando la comprensión de conceptos complejos. Se incluyeron guías paso a paso para la instalación de herramientas y la configuración de entornos de desarrollo, lo que permitió aplicar los conocimientos de manera práctica. Además, se proporcionaron ejemplos concretos y referencias a recursos externos, lo que enriqueció el aprendizaje y ayudó a contextualizar la información en escenarios reales.
