El documento proporcionado abarca una amplia gama de temas relacionados con el desarrollo de aplicaciones web utilizando tecnologías Java, específicamente enfocadas en JSP (Jakarta Server Pages), Spring Framework, y Spring Boot. También se mencionan herramientas como Apache Tomcat para el despliegue de aplicaciones y Maven para la gestión de dependencias. Además, se incluyen guías prácticas para la instalación y configuración de entornos de desarrollo, así como ejercicios para implementar APIs RESTful con Spring.

¿Qué aprendimos?

  • Apache Tomcat: Aprendimos que Tomcat es un contenedor de servlets que implementa las especificaciones de Java Servlet y JSP. Exploramos su estructura de directorios y el archivo de configuración server.xml, que define parámetros clave como puertos, hilos y timeouts.
  • Gestión de Dependencias con Maven: Maven es una herramienta para gestionar proyectos Java mediante el archivo pom.xml, que define dependencias, plugins y configuraciones del proyecto. Vimos cómo Maven facilita tareas como compilación, empaquetado y despliegue.
  • Arquitectura MVC en JSP: El patrón Modelo-Vista-Controlador (MVC) separa la lógica de negocio (Modelo), la presentación (Vista) y el control de flujo (Controlador). Vimos ejemplos de cómo implementar MVC en aplicaciones JSP.
  • Spring Boot: Spring Boot simplifica la configuración y el despliegue de aplicaciones Spring, incluyendo un servidor embebido. Aprendimos a crear proyectos con Spring Initializr y a estructurar aplicaciones con capas como controllers, models, repositories y services.
  • Desarrollo de APIs RESTful: Implementamos operaciones CRUD para entidades como Estudiante y Docente, utilizando Spring Data JPA y MySQL. Exploramos endpoints avanzados, validaciones, manejo de excepciones y documentación con Swagger.

¿Cómo lo aprendimos?

1. Teoría y Explicaciones Conceptuales: El documento proporciona definiciones claras de cada tecnología y concepto, acompañadas de diagramas y ejemplos.

2. Ejemplos Prácticos: Se incluyen fragmentos de código y configuraciones (como pom.xml, server.xml y clases Java) que ilustran cómo aplicar los conceptos en proyectos reales.

3. Guías Paso a Paso: Instrucciones detalladas para instalar herramientas como JDK, Tomcat, y configurar entornos de desarrollo en IntelliJ IDEA.

4. Ejercicios y Prácticas Calificadas: Los ejercicios propuestos (como la creación de APIs para gestión de docentes o estudiantes) permiten aplicar lo aprendido en escenarios realistas.

5. Integración de Herramientas: Uso combinado de Maven, Spring Boot, JPA y MySQL para desarrollar aplicaciones completas, desde el backend hasta la interacción con la base de datos.
