El documento es una guía práctica sobre el desarrollo backend con PHP y Laravel, enfocada en la configuración de herramientas esenciales como Apache HTTP Server, PHP, Composer y el framework Laravel. Incluye desde conceptos básicos hasta la implementación de un proyecto funcional con conexión a bases de datos, migraciones, rutas, controladores y vistas.

¿Qué aprendimos?

Configuramos Apache HTTP Server como servidor web local.

Instalamos y configuramos PHP, habilitando extensiones clave.

Usamos Composer para gestionar dependencias e instalar Laravel.

Creamos un proyecto Laravel con autenticación y estructura MVC.

Desarrollamos un CRUD básico para una entidad Estudiante, incluyendo:

  • Migraciones para la base de datos.
  • Rutas y controladores.
  • Vistas con Tailwind CSS.

Ejecutamos el servidor de desarrollo y probamos la aplicación.

¿Cómo lo hicimos?

  • Apache:
    • Descargamos la versión portable de Apache Lounge.
    • Editamos httpd.conf para definir puertos (Listen 8080) y rutas.
    • Ejecutamos el servidor con httpd.exe.
  • PHP:
    • Descargamos el ZIP oficial y lo configuramos en php.ini.
    • Agregamos PHP al PATH del sistema y verificamos con php -v.
  • Composer:
    • Lo instalamos globalmente y creamos un proyecto Laravel con:
    • composer create-project laravel/laravel estudiantes-app
  • Laravel:
    • Configuramos MySQL en el archivo .env.
    • Generamos migraciones con php artisan make:model Estudiante -m.
    • Creamos rutas en web.php, controladores (EstudianteController) y vistas con Blade.
    • Usamos Tailwind CSS para el diseño y compilamos assets con npm run dev.
    • Levantamos el servidor con php artisan serve.
