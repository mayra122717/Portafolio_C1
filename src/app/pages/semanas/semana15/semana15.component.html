El documento es una guía metodológica para el desarrollo de sistemas inteligentes utilizando tecnologías como Python, Flask y Django. Abarca desde conceptos teóricos (definición de sistemas inteligentes, arquitecturas, redes neuronales, deep learning, algoritmos de toma de decisiones) hasta ejercicios prácticos, como la implementación de un sistema experto basado en reglas IF-THEN, un controlador difuso de climatización y una interfaz web para interactuar con el modelo de lenguaje Llama 3.

¿Qué aprendimos?

Durante este proceso de aprendizaje, comprendimos los fundamentos y alcances de los sistemas inteligentes, incluyendo su capacidad para percibir el entorno a través de sensores, razonar simbólicamente mediante estructuras lógicas, y aprender de manera adaptativa con base en datos e interacciones.

Estudiamos la arquitectura general de los sistemas inteligentes, reconociendo los módulos principales: percepción, base de conocimiento, razonamiento, aprendizaje y acción. Esta estructura modular permite a los sistemas tomar decisiones autónomas, adaptarse a cambios y mejorar su desempeño con el tiempo.

Exploramos diversos tipos de redes neuronales, como las perceptronas multicapa (MLP), redes convolucionales (CNN), recurrentes (RNN) y Transformers, destacando su papel clave en el aprendizaje profundo (deep learning) y su aplicación en visión por computadora, procesamiento del lenguaje natural y predicción de secuencias.

Además, abordamos algoritmos de toma de decisiones como los sistemas basados en reglas, la lógica difusa y los árboles de decisión, que permiten a los sistemas responder a escenarios inciertos o ambiguos.

Finalmente, distinguimos entre modelos discriminativos, que aprenden la frontera de decisión entre clases (como SVM o regresión logística), y modelos generativos, que aprenden la distribución conjunta de los datos (como Naive Bayes o GANs), analizando ejemplos prácticos de cada uno.

¿Cómo lo aprendimos?

  1. Enfoque teórico-práctico:
    • Explicaciones conceptuales seguidas de ejercicios aplicados (ej: diagnóstico médico, control de climatización).
    • Ejemplos concretos de sistemas inteligentes en áreas como salud, automoción, retail, entre otros.
  2. Metodología paso a paso:
    • Instrucciones detalladas para configurar entornos, instalar dependencias y ejecutar código.
    • Estructura clara de proyectos con división entre backend, frontend y gestión de modelos de IA.
  3. Herramientas interactivas:
    • Uso de Jupyter Notebooks o scripts de Python para probar algoritmos de forma inmediata.
    • Desarrollo de interfaces web dinámicas (HTML/CSS/JS + Flask) para interactuar con modelos inteligentes.
  4. Evaluación y reflexión:
    • Preguntas de repaso para afianzar los conocimientos adquiridos.
    • Ejercicios conceptuales como diferenciar entre AGI (inteligencia general) e IA débil, o analizar aplicaciones de modelos LLM.
