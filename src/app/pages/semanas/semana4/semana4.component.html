HTML (HyperText Markup Language) es el lenguaje de marcado fundamental para la creación de páginas web, que permite estructurar y presentar contenido de manera organizada. En su nivel básico, HTML incluye elementos como etiquetas para definir la estructura del documento (html, head, body), formatear texto (p, h1 a h6, strong, em), crear listas (ul, ol), insertar imágenes (img), y enlaces (a). También cubre la creación de tablas (table) y formularios (form), así como el uso de comentarios y etiquetas semánticas (header, footer, main, section, article). En el nivel avanzado, HTML se expande para incluir atributos globales, contenido dinámico mediante templates y slots, elementos multimedia, gráficos con SVG y Canvas, formularios avanzados, y APIs de HTML5 que mejoran la interactividad y la accesibilidad, así como prácticas de SEO para optimizar la visibilidad en motores de búsqueda. En conjunto, estos temas proporcionan una base sólida para el desarrollo web moderno.

¿Qué aprendimos?

Aprendimos los fundamentos de HTML básico, como la estructura de un documento HTML, el uso de etiquetas esenciales (como html, head, body, h1, p, a, img, etc.) y cómo organizar el contenido en una página web. A nivel avanzado, exploramos el uso de tablas, formularios, etiquetas semánticas (header, nav, section, footer) y la integración con hojas de estilo externas. También conocimos buenas prácticas para escribir código limpio, estructurado y accesible.

¿Cómo lo aprendimos?

Lo aprendimos mediante una combinación de clases teóricas, ejemplos explicativos y ejercicios prácticos. Escribimos y ejecutamos código HTML directamente en el navegador y en editores como Visual Studio Code, lo que nos permitió ver resultados en tiempo real. Además, reforzamos el aprendizaje con prácticas que incluían estructuras más complejas, formularios interactivos y uso de atributos avanzados, facilitando una comprensión progresiva del lenguaje.
