Se desarrolló un sistema de login utilizando Node.js y Express para el backend, junto con React para el frontend. La autenticación se maneja a través de tokens JWT (JSON Web Tokens), lo que permite una gestión segura de sesiones. La base de datos utilizada fue PostgreSQL, asegurando una persistencia robusta de los datos de los usuarios. Este sistema permite a los usuarios registrarse, iniciar sesión y gestionar su sesión de manera segura.

¿Qué aprendimos?

Aprendimos sobre la implementación de sistemas de autenticación utilizando tokens, así como la integración entre el frontend y el backend. También profundizamos en el manejo de bases de datos relacionales con PostgreSQL.

¿Cómo lo aprendimos?

El aprendizaje se realizó a través de la investigación de documentación, tutoriales en línea y la práctica mediante la creación del proyecto. Trabajamos en equipo, lo que facilitó el intercambio de conocimientos y la resolución de problemas.
